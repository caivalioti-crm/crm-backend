\# Frontend v1 Scope Lock



\## ✅ IN SCOPE (v1)



\### Read

\- Customer dashboard KPIs

\- Customer readiness score

\- Top 3 categories per customer

\- Neglected categories (>90 days)

\- Full category \& subcategory discussion history (on-demand)



\### Write

\- Save visit intelligence

&#x20; - Categories discussed

&#x20; - Subcategories discussed

&#x20; - Visit date (dd/mm/yy)



\### UX

\- Lazy loading for non-critical data

\- In-memory caching per session

\- Cache invalidation on visit save



\---



\## ❌ OUT OF SCOPE (v1)



\- User authentication / roles

\- Offline mode

\- Editing or deleting visits

\- Historical visit timeline UI

\- Soft1 write-back

\- Analytics beyond current views

\- Cross-customer dashboards

\- Admin configuration screens

\- Background sync / jobs

\- Mobile optimization



\---



\## 🔒 SCOPE RULE



Any item not explicitly listed under \*\*IN SCOPE (v1)\*\*

is deferred to a later version.
